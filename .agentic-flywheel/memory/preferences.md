# Preferences Memory

Stable user preferences that should be applied by default when relevant.

## Writing

- Write blog post content in English unless the user explicitly asks for another language.
- Keep public posts sanitized and avoid company-specific confidential details.
- When adding sources to blog posts, place source links inline with the relevant claims when possible.
- For blog-post diagrams, prefer simple box-style ASCII diagrams over Mermaid unless Mermaid is explicitly requested.

## Collaboration

- Treat user corrections during iteration as potential reusable preferences, not only one-off edits.
- When a correction is likely to recur, record it in the harness after the task.
